# latent-perturbation-scRNA
Latent space perturbation analysis of gene effects in single-cell RNA-seq data
# Latent Perturbation scRNA Analysis

This repository contains code for the manuscript:

"Latent Space Perturbation Analysis Reveals Approximate Additive Gene Effects in Single-Cell Transcriptomic Dynamics"

## Overview
We analyze gene perturbations in latent space using single-cell RNA-seq data and evaluate approximate additivity of gene effects.

## Dataset
- PBMC dataset (pbmc68k_reduced) via Scanpy

## Requirements
- scanpy
- numpy
- matplotlib
- torch

## Usage
Run the pipeline:

python analysis_pipeline.py
